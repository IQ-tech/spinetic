<br />

## Getting Started

Welcome to Spinetic, a simple and customizable React carousel component.
This guide will walk you through the installation process and provide a brief introduction on how to use
Spinetic in your React applications.

## Installation

Using npm

```shell
npm i react-spinetic
```

Using yarn

```shell
yarn add react-spinetic
```

### Example Usage

1. Import the styles in your `main app.tsx` file:

```tsx
import "react-spinetic/styles.css";
```

2. Import the carousel components wherever you want to use them:

```tsx
import { Spinetic, SpineticItem } from "react-spinetic";
```

3. Use the following example to understand how to use Spinetic:

```tsx
function App() {
  const exampleItems = [
    "content-1",
    "content-2",
    "content-3",
    "content-4",
    "content-5",
  ];

  return (
    <Spinetic config={{ autoRotate: true }}>
      {exampleItems.map((content, index) => (
        <SpineticItem key={index}>
          <div
            style={{ height: 200, width: 250, background: "blue", margin: 10 }}
          >
            {content}
          </div>
        </SpineticItem>
      ))}
    </Spinetic>
  );
}

export default App;
```

This example demonstrates a basic setup of Spinetic. Customize appearance and behavior through configuration options and enjoy a integration of a carousel in your React project.

## `children`: _ReactNode_ or _ReactNode[]_

Children elements to be rendered inside the Spinetic component. It can be a single `ReactNode` or an _array_ of `ReactNodes`.

## `change`: _function_

Callback function triggered on a change event in the Spinetic component. The `change` function is called every time an item in the carousel is changed, providing information about the previous and current state.

- **Callback Signature**:
  ```tsx
  (event: SpineticChangeEvent) => void
  ```

- **`previous`**: Object containing information about the previous state.
  - `index`: The index of the previously active item.
  - `remainingIndexes`: The number of remaining items in the carousel.
  - `totalItems`: The total number of items in the carousel.

- **`current`**: Object containing information about the current state.
  - `index`: The index of the currently active item.
  - `remainingIndexes`: The number of remaining items in the carousel.
  - `totalItems`: The total number of items in the carousel.

### Example Usage:

```tsx
const handleChange = (event: SpineticChangeEvent) => {
  console.log('Previous State:', event.previous);
  console.log('Current State:', event.current);
};
```

The SpineticChangeEvent object is used within the change callback function, offering a structured way to access details about the previous and current states of the carousel during a change event.

```tsx
export type SpineticChangeEvent = {
  previous: {
    index: number;
    remainingIndexes: number[];
    totalItems: number;
  };
  current: {
    index: number;
    remainingIndexes: number[];
    totalItems: number;
  };
};
```
